HTML

  • Hypertext Markup Language(HTML) is used to create the structure of a webpage.
  • Made up of elements it is used to provide the content and basic framework for all webpages.
  • Elements seen by end user include Text, image, header, list.
  • Elements typically seen only by the dev include meta-related, content division and semantic.
  • Elements that are inside other elements are called child elements. The ones holding child elements are parent elements
  • Elements refer to the entire section from opening to closing tags. Tags refer to what's in the bracket
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • Cascading Style Sheets(CSS) is used to style a particular page element
  • components such as the color of a heading and a change to the rule will affect all tagged elements
  • Class attribute is an attribute used to define a group of HTML elements inorder to apply unique styling
  • and formatting to those elements with CSS
  • An internal CSS styles are included within the head seciton of an HTML doc and
  • apply to the entire doc allowing for consistent styling across multiple elements
  • External syles are stored in a separate file and can be linked to multiple HTML docs
  • allowing for global styling across an entire website
  • A margin indicates how much space we want around teh outside of an element.
  • A padding indicates how much space we want around the content inside an element
